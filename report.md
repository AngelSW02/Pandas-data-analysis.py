# Report: Pandas Data Analysis

## Dataset Description
The dataset contains six employees with their names, ages, salaries, and departments. It was created specifically for this assignment.

## Operations Performed
- CSV loading using pandas
- Displaying head(), info(), and describe()
- Accessing rows and columns using loc and iloc
- Boolean filtering with one and multiple conditions
- Adding and dropping columns
- Grouping data using groupby()

## Findings
The Finance department has the highest average salary. IT and HR follow behind. All numeric values appear within expected ranges.

## Limitations
This dataset is small and artificial, so it does not contain missing values or real real-world irregularities.
